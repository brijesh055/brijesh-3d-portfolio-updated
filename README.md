# Brijesh 3D Portfolio (React + Vite + Three.js)

## Run locally
```bash
npm install
npm run dev
```

## Projects section
Edit `data.projects` inside `src/App.jsx`:
- Add `github` and `demo` links.

## Google Sheet Contact Form
1. Create Google Apps Script Web App to save form data to Google Sheet.
2. Paste the Web App URL in `GOOGLE_SHEET_WEB_APP_URL` inside `src/App.jsx`.
